# Notification_system

Сервис для отправки уведомлений пользователям

## 📌 О проекте

Приложение позволяет запустить рассылку уведомлений через Email, Telegram и SMS.  
Для этого необходимо отправить POST-запрос со следующими данными: 
- Список пользователей 
- Тема сообщения
- Текст сообщения  

Приложение отправляет уведомления по электронной почте.  
В случае неудачи отправки по электронной почте, отправит через Telegram.  
В случае неудачи отправки через Telegram, отправит SMS.  
По окончании работы сформирует отчет и сохранит его в media/reports

Проект реализован на Django + DRF с поддержкой Docker и PostgreSQL

#### Пример данных post-запроса:
{
    "users": [
        ["Тамир", "tamirmandreev@mail.ru", "+79915410704", "tg_chat_id"],
        ["Тимур", "mandreevts@gmail.com", "+79644111469", "tg_chat_id"]
    ],
    "subject": "Тестовое письмо",
    "message": "Проводится тестирование создания объекта модели Notification"
}


## 🚀 Установка и запуск
1. Создайте файл .env, скопируйте в него содержимое файла .env.example и заполните значение переменных
2. Запустите docker compose
```bash
docker compose up -d
```
